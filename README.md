ajax.js是一个轻量级ajax功能插件
====


## post

ajax,post请求接口

```javascript
AJAX.post('app.php',{'data':'app'},function(data){/*-------*/})
```
## get

ajax,get请求接口

```javascript
AJAX.get('app.php?data=app',function(data){/*-------*/})
```
## getJSON

ajax,跨域请求接口

```javascript
AJAX.getJSON('app.php',{'data':'app'},function(data){/*-------*/})
```
